# florp

Markdown Browser Viewer

## System Wide Installation

```
$ poetry install
$ poetry build
$ pipx install dist/florp-0.1.0-py3-none-any.whl
```

## Usage

```
$ florp <filename.md>
```

## Dev setup

```
$ poetry install
```

### To run
```
$ poetry run florp <filename.md>
```
Or after running `poetry shell`:
```
$ florp <filename.md>
```
